# 🚀 Todo HTTP Server (Node.js)

A simple backend project built using **pure Node.js (no Express)**.

This project demonstrates how to create a REST-like API using the built-in HTTP module and file system.

---

## 📌 Features

- ➕ Add Todo (POST)
- 📋 Get All Todos (GET)
- ✔️ Mark Todo as Done (PUT)
- 🗑️ Delete Todo (DELETE)
- 💾 Data stored in JSON file (file-based database)

---

## 🛠️ Tech Stack

- Node.js
- HTTP Module
- File System (fs)
